Ecotrack
        is a sustainable lifestyle web application designed to help users track their eco-friendly challenges, measure their environmental impact, and join a global community of changemakers working toward a greener future.

Live Site URL:https://ass-10ph.netlify.app

. Track Green Challenges: Join, complete, and monitor your progress in various eco-friendly challenges such as tree planting, or energy saving.
. Personalized Dashboard: Users can view their activities, completed tasks, and overall impact.
. Secure Authentication: Firebase-based user login and registration system with protected routes.
. Responsive Design: Built with Tailwind CSS and DaisyUI for a modern, mobile-friendly interface.
. Dynamic Navigation: Smart navbar with user-specific options and smooth navigation using React Router.
. Real-time Notifications: Toast alerts for user actions like login, logout, or challenge completion.
